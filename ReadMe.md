# Aorta CT Stent Detection and Analysis

Python-based medical image processing pipeline for detecting and analyzing aortic stents from CTA scans using image processing and deep learning techniques.

## Features

- CTA/DICOM image loading
- Aorta segmentation
- Stent detection and tracking
- Image preprocessing and enhancement
- 3D medical image analysis
- Deep learning integration

## Technologies

- Python
- OpenCV
- NumPy
- PyTorch
- pydicom
- SimpleITK
- scikit-image

## Research Focus

This project focuses on cardiovascular CT image analysis and AI-assisted medical imaging for aortic assessment.

## Applications

- Medical image analysis
- Computer vision research
- AI-based healthcare systems
- Cardiovascular imaging
- Stent monitoring
